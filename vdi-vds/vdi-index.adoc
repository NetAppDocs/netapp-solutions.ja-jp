== ネットアップの仮想デスクトップサービス（ VDS ）

ネットアップの仮想デスクトップサービス（ VDS ）は、主要なパブリッククラウドとプライベートクラウドで Remote Desktop Services （ RDS ）のオーケストレーションを実現します。
